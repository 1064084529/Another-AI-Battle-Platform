### 胜利目标：<br>
    蛇的头部撞到障碍物，或者是撞到我方/敌方蛇的身体就算输，所以您只需要保证对手的失误在您之前即可。

    手动操作：按键盘WASD操控蛇上下左右移动,游戏开始时在对战地图下方会告知您位于左下方还是右上方。
